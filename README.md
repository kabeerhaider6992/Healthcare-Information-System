# Clinic Information Management System

A full-featured desktop-based application built using Python and SQLite for managing clinic operations efficiently. This system helps in managing patient records, medicine inventory, diagnosis history, and generating printable reports in a structured and organized way.

---

## 🚀 Features

- Patient registration and management
- Medicine inventory management
- Patient medical history tracking
- Advanced search and filtering system
- Date and time-based record filtering
- Auto-suggestions for medicines and diagnosis input
- Report generation system
- Printable medical reports and prescriptions
- Dark and light theme support

---

## 🛠️ Technologies Used

- Python
- SQLite Database
- Tkinter (GUI)
- OOP (Object-Oriented Programming)

---

## 📂 Project Structure

- `main.py` → Main application file containing all functionality
- `database.db` → SQLite database file
- `README.md` → Project documentation
